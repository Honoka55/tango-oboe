# tango-oboe
 A simple console application for reviewing Japanese words.
***
### Tsukaikata 
* 在 tango-risuto.txt 中记录已经背过的单词，每行一条，表记、假名、中文，以制表符隔开。
* 可以使用“复习读音”和“复习含义”两种模式进行复习。
